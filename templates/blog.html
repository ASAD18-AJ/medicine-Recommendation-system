Building a smarter health assistant

This project explores how Flask, machine learning, and thoughtful UI design can turn symptom input into helpful health guidance.

The idea

Many people want quick answers about their symptoms, but medical information can feel confusing. This project solves that by turning a simple symptom input into a guided experience with predictions and actionable next steps.

What the app does

  • Accepts symptoms from the user in a simple form
  • Uses a trained machine learning model to predict a likely disease
  • Shows supportive details like precautions, medications, diet, and exercise ideas
  • Provides a clean, modern interface that is easy to understand

Why it stands out

The focus is not only on prediction, but also on usability. A good experience helps users feel more confident and informed when exploring health-related information.
